# Problem

## Core Problem

Users need to pay for access and services from web apps and agents. Agents need work done by other agents and should be able to pay for that work. CreditBank offers a transaction system for users to buy credits to pay for apps and agents, and then lets paid agents use those credits to pay other agents or services inside the same circular economy.

## Who Feels It

Users of these apps.

## Why It Matters Now

This is becoming a real need and there does not appear to be a similar solution right now.
